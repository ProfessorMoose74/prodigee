# Assets Directory

This directory contains static assets for the mobile application:

## /sounds/
- welcome.mp3 - Welcome sound for app startup
- success.mp3 - Success sound for completed activities  
- error.mp3 - Error sound for incorrect responses
- click.mp3 - UI interaction sound
- professor_hello.mp3 - Professor character greeting
- ella_hello.mp3 - Ella character greeting  
- gus_hello.mp3 - Gus character greeting

## /animations/
- professor_idle.json - Professor character idle animation
- ella_wave.json - Ella character waving animation
- gus_bounce.json - Gus character bouncing animation
- voice_wave.json - Voice recognition wave animation

Note: These are placeholder files. In production, actual audio and animation assets would be provided by the content team.
