# Content Directory Structure

This directory contains educational content files for the Elemental Genius platform:

## Structure:
- /phonemic/ - Phonemic awareness activities and audio files
- /math/ - Mathematics learning content by age group
- /science/ - Science exploration materials
- /astronomy/ - Space and astronomy content
- /geography/ - World geography learning materials

## File Types:
- Audio files (.mp3, .wav) for pronunciation and instructions
- Animation files (.json) for Lottie animations  
- Image files (.png, .jpg) for visual learning aids
- Video files (.mp4) for interactive lessons

Note: This is a placeholder structure. In production, content would be managed through a proper CMS system.
