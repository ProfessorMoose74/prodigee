# Perplexity Linux

## Install Prerequisists
```
sudo zypper install python3-qt6 python3-requests
```

## Step 4: Run the Application
Add Your API Key: Open the perplexity_kde_app.py file and replace "YOUR_PERPLEXITY_API_KEY" with the actual key you copied.

Save the file.

Open Konsole and navigate to the directory where you saved the file. For example, if it's on your Desktop:
```
bash
cd ~/Desktop
Run the script:

bash
python3 perplexity_kde_app.py
```
A native window should appear on your desktop!
